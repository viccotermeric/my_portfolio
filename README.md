# Rishabh Trivedi - Interactive Terminal Portfolio

Welcome to my personal developer portfolio! This is an interactive, terminal-style web application built to showcase my backend engineering, data architecture, and full-stack projects. 

![Portfolio Front Page](public/og-image.png)

## Tech Stack
- **Frontend**: React & Vite
- **Styling**: Tailwind CSS
- **Deployment**: Vercel

## Overview
All portfolio data (such as experience, education, and projects) is rendered dynamically through an emulated command-line interface. Feel free to explore the interactive terminal or switch using the GUI toggle!
